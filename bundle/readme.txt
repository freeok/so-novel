Windows 用户
运行 SoNovel.exe

macOS 用户
运行 run-macos.sh

Linux 用户
运行 run-linux.sh


温馨提示
- 为获得最佳使用体验，请将终端窗口最大化。
- config.ini 是配置文件，每个配置项有对应的注释，修改保存后无需重启应用（非 Windows 不保证）。
- 如果认为下载速度较慢，适当减小爬取间隔可能有助于提高速度，直到达到合适的平衡。
- 设置过小的爬取间隔会导致部分数据源封禁 IP，从而无法使用。
- 如果目标内容的标题搜不到，就用作者或发布者名称搜，反之亦然。
