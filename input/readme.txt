Windows 用户
运行 exe

macOS 用户
运行 macos-run.sh

Linux 用户
运行 linux-run.sh


使用须知（必读）
1. 请按要求操作，然后按 Enter 键执行，若不按要求操作则无法正常工作
2. 若无法搜索，建议多次尝试、稍后再试、检查网络环境
3. 若章节下载失败，可尝试增大爬取间隔，直至合适为止
4. 若认为下载速度较慢，可适当减小爬取间隔，直至合适为止
5. 爬取间隔过小可能会导致 IP 被部分书源限流，短时间内将无法使用


温馨提示
1. 将终端窗口最大化，使用体验更佳
2. config.ini 是配置文件，书源、下载格式在此修改，修改后无需重启（不保证非 Windows 适用）
